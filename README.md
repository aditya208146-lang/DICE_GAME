🎲 Dice Game (Python)

A simple Dice Game built using Python that allows players to log in, roll dice, and view results.
This project demonstrates modular programming, basic game logic, and user interaction through a console-based interface.


📂 Project Structure
Dice_Game/
│── main.py        # Entry point of the game
│── __init__.py    # Initializes the package
│── login.py       # Handles player login/authentication
│── player.py      # Manages player data and actions
│── result.py      # Determines and displays game results

🚀 Features
🎮 Simple and interactive dice game
👤 Player login system
🎲 Random dice rolling mechanism
🏆 Result calculation (win/lose conditions)
🔁 Replay option for multiple rounds
🔐 Basic validation for user input


🛠️ Technologies Used
Python 3.x
Random module
Modular programming

▶️ How to Run

1---Clone the repository:

https://github.com/aditya208146-lang/DICE_GAME

2---Navigate to the project folder:

cd dice-game

3---Run the game:

python main.py


🧠 How It Works
-main.py controls the game flow and menu
-login.py handles player authentication
-player.py stores and manages player details
-Dice rolls are generated using Python’s random functions
-result.py compares outcomes and declares the winner
